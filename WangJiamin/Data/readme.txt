# Instructions

1. Download the dataset from Kaggle:  
   [https://www.kaggle.com/datasets/carlosgdcj/genius-song-lyrics-with-language-information/data](https://www.kaggle.com/datasets/carlosgdcj/genius-song-lyrics-with-language-information/data)

2. Save the file **`song_lyrics.csv`** into:  
   ```plaintext
   ./data/song_lyrics/song_lyrics.csv
   ```

3. Ensure the dataset contains song metadata (title, tag, artist, year, lyrics) and language detection fields.

4. Only English songs from **1960 to 2019** will be used during preprocessing.

5. Follow the provided R script to preprocess, filter, sample, and clean the data.
